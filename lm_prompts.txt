## LM Studio prompts for Phase 3 image/chart zone recognition.
## Users can edit these prompts to customize behavior for their documents.
## Sections start with a single # section_name.
## Valid section names: image, image_modification, chart

# image
Si c'est un circuit, nomme les composants avec leurs valeurs si visibles. Sinon, décris brièvement. Saute si c'est une ligne ou un élément sans intérêt. Pas d'introduction, juste le contenu.

# image_modification
Cite uniquement les composants modifiés avec leurs nouvelles valeurs si visibles. Pas d'introduction.

# chart
Décris le graphique en une phrase : axes, tendance, valeurs clés si visibles. Pas d'introduction.
